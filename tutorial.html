Bienvenido al sistema ATP Tracker

Objetivo

Durante el curso de Desarrollo Web hice un sitio web para un gimnasio llamado ATP, durante el curso de JavaScript quiero generar una aplicación que permita registrar a los usuarios y sea una forma de consultar los avances de cada persona. La aplicación podría ser utilizada tanto por los miembos del gimnasio como por el staff.

Los miembos podran ver las rutinas que tienen asignadas y sus progresos: peso, grasa corporal, músculo, etc. Mientras que los miembros del staff podran observar los mismos datos y datos adicionales como las fechas de pago de cada usuario

Funcionamiento

Para esta primera entrega hice una aplicación en donde se pueden agregar usuarios al gimnasio, modificar los datos de los usuarios existentes, consultar los datos de los usuarios y por último imprimir todos los datos de los usuarios en la consola. El código ya viene con 2 usuarios registrados para facilitar las pruebas

Opción 1: Sirve para agregar usuarios a la base de datos del gimnasio. La aplicación solicita los siguientes datos: Nombre, Edad, Peso, Sucursal del gimnasio donde estará, Rutina de ejercicios que realizará

Opción 2: Sirve para consultar y actualizar la información de algún miembro. Se puede actualizar el peso, la sucursal y la rutina. Al ingresar a esta opción se solicita un nombre para buscar en la base de datos

Opción 3: Revisa el avance del usuario en el gimnasio, para esto compara el peso anterior con el peso actual. Si el peso es mayor se sugiere al usuario que pase con el entrenador para ajustar la rutina y con la nutriologa a una revisión. Si el peso actual es menor se felicita al usuario

Opción 4: Imprime en la consola todos los usuarios junto con los valores que tienen

Opción 5: cierra el programa
